# 性能基线

## 目标指标
1. 滚动流畅度：长对话滚动体感明显改善。
2. DOM 规模：挂载消息数不超过 `maxMountedMessages + overscan` 的量级。
3. 资源占用：CPU 与内存增长趋势低于未开启扩展场景。

## 场景定义
- 会话长度：> 200 条消息。
- 操作方式：持续滚动 30 秒（向上、向下各一次）。
- 对照组：扩展关闭。
- 实验组：扩展开启，默认参数（80/8/6）。
